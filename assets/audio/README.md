# Audio Assets

Everything in `sfx/` and `bgm/` is a procedurally synthesized placeholder generated by `tools/generate_placeholder_audio.py`.
Regenerate at any time with `python3 tools/generate_placeholder_audio.py` (requires numpy).

These placeholders exist so the full SFX event matrix and BGM cross-fade from `docs/juice_manual.md` section 5 are audible and testable end-to-end.
They should be replaced with sourced assets per section 6 (Sonniss GDC, OpenGameArt CC0, itch.io packs).
When replacing, keep the exact filenames - `AudioManager.gd` maps events to these paths.
`AudioManager` forces loop flags on BGM streams at runtime, so replacement BGM only needs to be a clean 30-45s loop.
